🔌 API Endpoints

🏆 Leaderboard Endpoints
GET /leaderboard - Get all players
GET /leaderboard?telegram_id=eq.{id} - Get specific player
GET /leaderboard?order=level.desc - Get top players by level
PATCH /leaderboard?telegram_id=eq.{id} - Update player data
🔗 Referral Endpoints
GET /referrals - Get all referrals
GET /referrals?referrer_telegram_id=eq.{id} - Get referrals by referrer
POST /referrals - Create new referral
GET /pending_referrals - Get pending referrals
💡 Query Examples
# Get top 10 players by level
/leaderboard?order=level.desc&limit=10
# Get referrals for specific user
/referrals?referrer_telegram_id=eq.123456789
# Get players with TAMA > 1000
/leaderboard?tama=gt.1000
